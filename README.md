## Instalação

Instale mitz-nft-nuxt na sua máquina

```bash
# yarn
yarn install

# npm
npm install
```

Inicia o projeto em ambiente de desenvolvimento

```bash
# Run the tests
$ yarn dev
# or
$ npm run dev
```
    
Rode os testes unitários

```bash
# Run the tests
$ yarn test
# or
$ npm run test
```
